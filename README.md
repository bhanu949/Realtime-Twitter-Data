"# Realtime-Twitter-Data" 

This App lets you get the tweets based on the search.

Installations Required (Back-End)
1) npm init -y
2) npm express
3) npm socket.io
4) npm twitter
5) npm nodemon

Front-End
1) npx create-react-app client
2) npm react-router-dom
3) npm react-redux
4) npm redux
5) npm thunk
6) npm axios
